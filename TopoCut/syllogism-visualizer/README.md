# 古典修辞三段论可视化构建工具

## 项目概述

基于 Svelte + SVG 开发的古典修辞三段论可视化构建工具，支持命题输入、格与式选择、论证有效性验证、逻辑谬误识别等功能。

## 核心功能

### 1. 三段论矩阵可视化
- 支持4个格的可视化展示
- 大前提、小前提、结论的命题输入
- 词项（S小项、P大项、M中项）的高亮标识
- 周延性分析展示

### 2. 四个预设谬误按钮
- **预设一：肯定后件谬误** - 形式谬误，P→Q, Q ⊢ P
- **预设二：四项错误** - 三段论中出现四个不同词项
- **预设三：存在预设陷阱** - 从全称前提推出特称结论
- **预设四：连锁悖论** - 多段论连锁导致的悖论

### 3. 动画效果
- **弹性绳索动画** - 命题间连接的弹性绘制效果
- **绿色光流传导** - 有效论证时的推理路径光流
- **红色电弧闪烁** - 无效论证时的矛盾爆发效果
- **圆环扩散动画** - 验证时的涟漪扩散效果
- **打字机墨迹动画** - 结论生成的逐字显示效果

### 4. 高级功能
- **词项周延检测** - 分析主项和谓项的周延性
- **语义歧义识别** - 检测多义词的不同使用
- **嵌套命题解析** - 支持复杂复合命题的解析栈保护
- **循环依赖检测** - 检测论证网络中的循环论证
- **谓词逻辑树** - 可视化展示命题间的逻辑关系

## 使用说明

1. **选择格** - 点击第1-4格按钮选择三段论的格
2. **输入命题** - 在大前提、小前提、结论输入框中输入命题
3. **选择命题类型** - 使用下拉框选择A/E/I/O四种命题类型
4. **验证论证** - 点击"验证论证有效性"按钮进行验证
5. **加载预设** - 点击左侧预设按钮直接加载经典谬误案例
6. **查看逻辑树** - 点击"显示逻辑树"按钮查看谓词逻辑分析

## 三段论基础知识

### 命题类型
- **A** - 全称肯定：所有S是P
- **E** - 全称否定：所有S不是P
- **I** - 特称肯定：有些S是P
- **O** - 特称否定：有些S不是P

### 四个格
- **第1格**：中项在大前提作主项，在小前提作谓项
- **第2格**：中项在两个前提中都作谓项
- **第3格**：中项在两个前提中都作主项
- **第4格**：中项在大前提作谓项，在小前提作主项

### 有效式
- 第1格：AAA, EAE, AII, EIO, AAI, EAO
- 第2格：AEE, EAE, AOO, EIO, AEO, EAO
- 第3格：AAI, IAI, AII, EAO, OAO, EIO
- 第4格：AAI, AEE, IAI, EAO, EIO, AEO
